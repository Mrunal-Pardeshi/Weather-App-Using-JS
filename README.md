# 🌤️ Weather App

A modern and responsive weather application built using **HTML, CSS, and JavaScript**. The app fetches real-time weather data from the **OpenWeatherMap API** and displays detailed weather information for any city in a clean and user-friendly interface.

## 🌐 Live Demo

🔗 **https://mrunal-pardeshi.github.io/Weather-App-Using-JS/**

> Replace the link above with your actual GitHub Pages URL if it's different.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature
- ☁️ Weather condition with dynamic icon
- 💧 Humidity
- 💨 Wind speed
- 🌡️ Feels like temperature
- 📊 Atmospheric pressure
- 👀 Visibility
- ⌨️ Search using the **Enter** key
- 📱 Fully responsive design
- 🎨 Modern and clean user interface

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
